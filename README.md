# where-was-i

## 개요

ADHD 환자 혹은 꾸준한 집중과 과업에 대한 추적이 필요한 사용자를 위한 비서봇

## 목표

- 사용자가 하루에 대한 계획을 수립하고 이를 성공적으로 수행하는 것을 돕는다.
- 계획의 성공적인 수행을 돕기 위해, 계획의 실행에 대해서 꾸준히 추적하고 기록할 수 있도록 돕는다.
- 지속적인 계회 수행능력 개선을 위해, 하루의 계획에 대해 회고하고 수행 중 발생한 변경이나 예외상황에 대한 기록과 관리를 가능케한다.

## 주요 기능

### 매일 매일 계획

- 매일 일과 전 정해둔 시간에 하루 계획에 대해서 정리하기
    - 사용자가 이전에 완수하지 못한 과업, 주기적인 과업, 장기 과업에 대해서 리마인드하고, 오늘 계획에 포함할지 함께 결정하기
    - 사용자가 말해준 항목들에 대해서 과업 단위로 정리하기
    - 너무 목표가 큰 과업을 세부 과업 형태로 분리하기
    - 목표가 불분명한 과업을 함께 구상화하기
    - 오늘 수행할 과업 확정하기

### 일과 중 기록과 주기적인 추적

- 일과 중에 지금부터 어떤 과업을 수행할지 기록하기
- 계획에 없던 과업이라면 새로 생성한 뒤 수행중으로 기록하기
- 과업 기록에 앞서 이전에 수행중이던 과업이 있다면, 완료된 것인지 단순 변경인지 확인하기
- 단순 변경이라면 필요한 경우 변경 사유를 기록하기
- 사용자가 정해놓은 주기에 따라 주기적으로 사용자에게 과업 기록을 요청하기

### 매일 매일 회고

- 매일 일과 후 정해둔 시간에 하루 계획 수행에 대해서 회고하기
    - 일과 중 수행한 과업의 완료 상태에 대해서 다시한번 체크하기
    - 완료하지 못한 과업들을 확인하고 필요한 경우 메모 남기기
    - 계획하지 않았지만 생성된 과업들에 대해서 확인하고 필요한 경우 메모 남기기
